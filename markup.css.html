Quick Setup

  1. Link the stylesheet: (self updating) <link href="https://acei.us/files/markupcss/latest.css" rel="stylesheet" type="text/css" /> in the <head> tags of your document
  2. Use the class= atribute on elements as needed (i.e. using the <span> element)
  3. Reference the guide as needed

Example HTML file:

<!DOCTYPE html>
<html>
<head>
<link href="https://acei.us/files/markupcss/latest.css" rel="stylesheet" type="text/css" />
</head>
<body>
</body>
</html>

Getting Started

(a.k.a. Quick Setup but longer and more detailed

Step 1

Before you use the stylesheet, you have to link it. You can link the minified latest sheet from my site, using this code snippet: <link href="https://acei.us/files/markupcss/latest.css" rel="stylesheet" type="text/css" />
Or, you can download the latest release from the downloads section of this page, or you can grab the latest pre-release (not reccomended) from GitHub.
Once you have downloaded the stylesheet, use the <link> element in your document <head>.

Step 2

Now that you have the stylesheet is linked, you can start applying mk- classes to elements on a page. For example, on headers you can use the mk-title class on header elements, to center and bold them.
Some more ideas you could do are using the mk-bold and mk-itallic classes for bold and itallic instead of <b> and <i>. The <b> and <i> elements are not screenreader friendly. You should not use them, and instead use css for decorative emphasis.
If you need to 'redact' text, you could use the mk-highlight-blackout class to have a blackout appearance. Of course, you should never use this for actual redacting, as the text can be revealed by turning off css.

Step 3

If you need more help, you can ask for help in my discord server, or look at the complete reference avalible below.
